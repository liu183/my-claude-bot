## 📊 角色分类总览

### 分类标准

根据角色的重要性、美貌等级、戏份占比和团队价值，将后宫角色分为三类：

1. **核心后宫**：5人，必须有独立戏份，推动剧情发展
2. **重要配角**：15人，有特殊作用，戏份相对较少
3. **普通成员**：30人，功能单一或边缘角色

### 后宫成员统计（第1-500章）

**总人数：50人**
- 核心后宫：5人
- 重要配角：15人
- 普通成员：30人

## 👑 核心后宫成员（5人）

### 1. 韩雪儿 - 青梅竹马

**基本信息**
- **姓名**：韩雪儿
- **年龄**：24岁
- **职业**：前世为程序员，末世后成为核心成员
- **外貌评分**：92分（S级）
- **性格特点**：
  - 温柔体贴，善解人意
  - 忠诚度最高（95%+）
  - 情感核心，第一个女人
  - 有隐藏身份（父亲是神秘组织首领）

**异能类型**
- 精神系异能（后期觉醒）
- 强大的感知能力
- 能预知危险

**加入过程**
- **章节**：序章+第1章
- **方式**：青梅竹马，前世恋人
- **难度**：★★☆☆☆（情感基础）
- **关键情节**：
  - 前世为救林天而死
  - 重生后第一个保护的人
  - 第280章揭露神秘身份

**在团队中的定位**
- **地位**：第一正宫，情感核心
- **职责**：
  - 内务管理
  - 情感纽带
  - 后宫和谐维护者
- **戏份占比**：15-20%

**经典场景**
1. 序章：奄奄一息时的告白
2. 第1章：重生后的第一眼
3. 第280章：身份揭露的信任危机
4. 第500章：完美结局
