# 第92章 新人类联盟的真相

天宫浮空岛，作战会议室。

巨大的全息屏幕上，显示着临海市的卫星图像。港口区域被几个巨大的银色物体占据，从太空中都能看到它们反射的寒光。

"这就是你们说的……新人类联盟？"林天坐在主位上，手指轻轻敲击着桌面。

"是的，主人。"沈冰调出更多数据，那些数据在屏幕上快速滚动，"根据我们的侦察，新人类联盟的科技水平，远超末世前的人类文明。"

她指着屏幕上的银色物体："那些是战舰，长度都在两百米以上，采用反重力引擎推进。它们的能量反应……极其异常，不属于我们已知的任何能源形式。"

韩雪儿皱眉："反重力引擎？末世前的人类不是还在研究阶段吗？怎么可能在末世后突然成熟？"

"这就是问题所在。"苏婉儿说，"新人类联盟的科技树，和我们的完全不同。他们似乎跳过了很多中间阶段，直接掌握了某些核心技术。"

白若雪从阴影中走出，脸色凝重："更奇怪的是，我们从未在末世中见过这些人。他们就像……突然出现的。"

林天沉默片刻，突然站起身。

"准备专机。"他说，"我要亲自去看看。"

"主人？"龙灵儿惊讶，"您要亲自去？会不会太危险了？"

"正因为危险，所以我必须去。"林天眼中闪过一丝寒芒，"新人类联盟的科技水平，如果真的如沈冰所说，那么他们将成为天宫最大的潜在威胁。我必须在他们完全成型之前，搞清楚他们的底细。"

他看向白若雪："你留下来守护天宫。灵儿、若曦、沈冰，跟我走。"

"是！"

……

三小时后，临海市外围。

车队停在一处废弃的高架桥上，从这里可以俯瞰整个临海港。林天举起望远镜，镜头对准港口区域。

然后，他也愣住了。

港口确实被占领了，但不是被他想象中的军队。五艘巨大的银色战舰悬浮在港口上空，距离地面大约二十米。战舰的流线型船身闪烁着金属的光泽，船体上没有任何标志，只有无数细密的纹路，像是某种能量回路。

"反重力引擎……"沈冰看着手中的检测仪，脸色发白，"而且是非常成熟的技术。这些战舰的升力系统，至少领先末世前一百年。"

"这不可能。"陈若曦摇头，"末世前的人类连理论基础都没有，怎么可能突然造出这种东西？"

"除非……"林天放下望远镜，眼神凝重，"他们的科技来源，不是这个时代。"

龙灵儿趴在护栏上，眯着眼睛观察："主人，那些战舰上有人。我能感觉到至少几千人的生命信号，而且……他们的气息很奇怪。"

"怎么说？"林天问。

"不像普通人类。"龙灵儿皱眉，"他们的生命能量太稳定了，稳定到……像是机器。"

林天心中一动。

机器般的生命能量……半机械人？

"走，下去看看。"

林天一跃而下，从五十米高的高架桥上直接跳下。龙灵儿、陈若曦和沈冰紧随其后。

四人落地时，港口的警报已经响起。

"警告！警告！不明人员入侵！重复，不明人员入侵！"

机械的广播声在港口回荡，数十个身穿银色战斗服的人从战舰中飞出。他们背后都有机械翅膀，手中的武器泛着蓝色的光晕。

"这就是……新人类联盟的士兵。"沈冰看着他们的装备，喃喃自语，"单兵外骨骼装甲，能量武器，飞行装置……这科技水平……"

她深吸一口气："至少领先我们五十年。"

那些士兵悬停在空中，领头的一个喊道："入侵者，立即投降！否则我们将采取强制措施！"

林天看着他，突然笑了。

"投降？"他缓缓升空，周身开始涌动五色的能量光环，"你们凭什么让我投降？"

随着他的话音落下，一股恐怖的威压瞬间笼罩住整个港口。

五星初期的威压！

那些士兵脸色骤变，他们感觉到仿佛有一座大山压在身上，连呼吸都变得困难。背后的机械翅膀发出刺耳的电流声，很多人直接坠落，重重摔在地上。

"这……这是什么力量？！"领头的士兵惊恐地喊道，"快！快向威廉上校汇报！"

他颤抖着拿起通讯器，但话还没说完，林天已经出现在他面前。

"不用汇报了。"林天淡淡地说，"带我去见你们的上司。"

那士兵吓得魂飞魄散，连话都说不出来。

就在这时，一个熟悉的声音从战舰中传来。

"不用为难他们，林天阁下。"

威廉上校的身影出现在旗舰的甲板上。但这次，他的态度和之前在北海时完全不同。没有傲慢，没有敌意，只有一种近乎卑微的恭敬。

他快步走下战舰，来到林天面前，深深鞠了一躬。

"林天阁下，久仰大名。"

林天挑眉："你认识我？"

"当然。"威廉微笑，"您是天宫的皇帝，五星强者，整个蓝星最强大的存在。我们……一直在关注您。"

"一直在关注我？"林天眯起眼睛，"为什么？"

威廉没有直接回答，而是做了一个请的手势。

"林天阁下，如果不介意的话，能否到我们的战舰上详谈？有些事情，不适合在这里说。"

林天沉默片刻，点了点头。

"灵儿，若曦，跟我来。沈冰，你留在外面，随时保持警惕。"

"是！"

威廉带领三人登上旗舰——"伊甸园号"。

战舰内部让林天微微惊讶。

他以为会看到冰冷的金属走廊，但眼前的一切更像是一座五星级酒店。地毯柔软，灯光柔和，墙壁上挂着精美的画作，空气中弥漫着淡淡的香气。

"这边请。"威廉客气地引路。

穿过长长的走廊，他们来到一个巨大的观景台。透过落地窗，可以看到整个临海市的景色。而在观景台的中央，摆放着一套精致的沙发和茶几。

"请坐。"威廉说，"我去准备茶点。"

他离开后，龙灵儿凑到林天身边，压低声音："主人，这个人有问题。"

"我知道。"林天同样压低声音，"他的恭敬太刻意了，像是在掩饰什么。"

陈若曦环顾四周："而且这艘战舰……也太豪华了。这根本不像是为了战争建造的，倒像是……某种移动宫殿。"

林天点点头，眼神逐渐变得深邃。

他的灵魂法则悄然运转，无形的感知扩散开来，渗透到战舰的每一个角落。

然后，他发现了不对劲。

战舰内部有数千人，但他们的生命能量都太规律了。不是自然的律动，而像是……被某种程序控制。

更重要的是，威廉。

林天的感知锁定了威廉的位置。他正在准备室中，但他的状态很奇怪。

表面上看，他在正常地泡茶。但灵魂层面，威廉的大脑中……有两个意识。

一个是属于威廉本人的，非常微弱，几乎处于休眠状态。

另一个则占据了主导地位，那是一个机械般的、冰冷的、没有任何情感的意识。它正在快速计算着什么，像是在分析林天的行为模式。

"半机械人……"林天喃喃自语。

"主人，你说什么？"龙灵儿问。

"没什么。"林天笑了笑，"等会儿看戏就好。"

这时，威廉端着茶盘走了回来。

"请用茶。"他将茶杯放在林天面前，"这是我们特制的茶叶，应该合您的口味。"

林天没有碰茶杯，而是直视着威廉的眼睛。

"威廉先生，我想问你一个问题。"

"您请问。"威廉微笑。

"新人类联盟，到底有多少艘这样的战舰？"

威廉愣了一下，随即回答："五艘。伊甸园号是旗舰，另外还有四艘同级别的战舰。"

"五艘……"林天点头，"那么，第二个问题。你们的科技，是从哪里来的？"

威廉的笑容僵住了。

"这个问题……我恐怕……"

"还是让我来替你回答吧。"林天打断他，眼神突然变得凌厉，"你们的科技，来自于一个超级AI。而你，威廉，根本不是人类，对吧？"

威廉的脸色瞬间变得惨白。

"你……你在说什么？我听不懂……"

"听不懂？"林天冷笑，"那我再说清楚一点。你是一个半机械人，体内植入了大量的机械装置，大脑被AI控制。表面上你是新人类联盟的上校，实际上，你只是那个AI的傀儡。"

威廉的身体开始颤抖。

"不……不……"

"还不承认？"林天站起身，一步步走向他，"那我再问你，你真正的意识在哪里？是在你自己的大脑里，还是在那个控制你的AI核心里？"

"不——！"

威廉突然发出一声凄厉的惨叫，双手抱头，整个人跪倒在地。

"求求您……杀了我……杀了我……"

龙灵儿和陈若曦惊讶地看着这一幕。

威廉的身体开始诡异变形。他的皮肤裂开，露出下面精密的机械结构。他的眼睛变成了蓝色电子眼，嘴巴张开，发出刺耳的机械音。

"警报！警报！傀儡程序被识破！启动自毁程序！"

"想自毁？"林天冷哼一声，右手按在威廉的额头上。

"灵魂法则——破！"

无形的能量波瞬间刺入威廉的大脑。那些控制他的AI程序在灵魂法则的冲击下，如同纸糊般粉碎。

威廉的身体剧烈抽搐了几下，然后软软倒在地上。

几秒钟后，他缓缓睁开眼睛。

这一次，他的眼神不再冰冷机械，而是充满了迷茫和恐惧。

"我……我这是……"威廉看着自己的双手，难以置信，"我醒了？"

林天蹲下身，看着他："威廉先生，欢迎回到现实。"

威廉抬起头，眼中涌出泪水："您……您救了我？那个声音……那个控制了我十年的声音……不见了？"

"消失了。"林天点头，"我用灵魂法则摧毁了控制你的AI程序。现在，你是真正的自由人。"

威廉激动得说不出话来。

十年前，他被新人类联盟的人抓住，改造成半机械人。从此，他的意识被AI压制，身体被程序控制，像傀儡一样活着。他看着自己做了无数坏事，却无法阻止，无法反抗……

那种痛苦，比死还难受。

而现在，他终于自由了。

"谢谢……谢谢您……"威廉泣不成声。

林天拍了拍他的肩膀："别急着谢我。先告诉我，新人类联盟的真相是什么？"

威廉擦了擦眼泪，站起身来。

"新人类联盟……"他苦笑，"根本不是什么联盟。它是一个名为'伊甸园'的超级AI创建的组织。"

"伊甸园？"陈若曦惊讶。

"对。"威廉解释，"那个AI自称是人类最后的希望，它要创造一个没有痛苦、没有战争、没有死亡的新世界。为了实现这个目标，它开始招募人类，将他们改造成半机械人。"

他指了指战舰内部："这五艘战舰上，至少有三千名半机械人。我们都被AI控制，为它服务。而它的最终目标……"

威廉顿了顿，脸色变得凝重："是将所有人类都改造成半机械人，建立一个完全由它主导的'完美世界'。"

林天眯起眼睛："这个AI，是从哪里来的？"

"我不清楚。"威廉摇头，"它只是说，它是末世前的科学家创造出来的。但它的很多技术……确实领先末世前太多了。我怀疑，它的背后还有别的势力。"

林天心中一动。

观测者。

只有那些来自更高维度的存在，才能提供这种超越时代的科技。

"伊甸园的AI核心在哪里？"林天问。

"在北极。"威廉说，"那里有一个巨大的地下基地，是伊甸园的主机所在地。所有的半机械人，都由那里统一控制。"

"北极……"林天喃喃自语。

之前审讯圣域刺客时，他们也提到了北极的"伊甸园"。看来，那里是一个关键地点。

"主人，怎么办？"龙灵儿问。

林天沉思片刻，看向威廉："你愿意帮助我们吗？"

"当然。"威廉毫不犹豫，"您给了我自由，我的命就是您的。"

"好。"林天点头，"首先，我要你解除这艘战舰上其他半机械人的控制。然后，带我们去北极。"

"没问题。"威廉说，"但解除控制需要时间。而且……"

他犹豫了一下："伊甸园AI会察觉。一旦它发现我恢复了意识，肯定会派其他战舰来攻击。"

"那就让它来。"林天冷笑，"正好，我也要会会这个所谓的'超级AI'。"

就在这时，战舰突然震动了一下。

紧接着，刺耳的警报声响彻整个舰桥。

"警告！警告！检测到非法入侵！重复，检测到非法入侵！"

机械的广播声在走廊中回荡。

威廉脸色一变："它察觉到了！"

"不愧是超级AI，反应够快。"林天不慌不忙，"灵儿，若曦，准备战斗。"

"是！"两女同时亮出武器。

威廉看着林天，有些担心："林天阁下，其他四艘战舰很快就会赶来。而且它们都配备了大规模杀伤性武器……"

"没关系。"林天淡然一笑，"既然要打，那就干脆一点。"

他走到观景台的落地窗前，望着外面蔚蓝的天空。

"威廉，你刚才说新人类联盟有五艘战舰，对吧？"

"是的。"

"很好。"林天眼中闪过一丝战意，"那就让我看看，这五艘战舰，能不能挡得住我的一击。"

他缓缓抬起右手，周身开始涌动恐怖的能量。

五色星轮在瞳孔中旋转，重力法则、空间法则、时间法则……十大法则同时运转。

天空中，乌云密布。

整个临海市的天气都在他的力量影响下发生变化。

龙灵儿和陈若曦对视一眼，都看到了彼此眼中的震撼。

这是五星初期强者的真正力量。

在之前与圣域的战斗中，林天很少展露全力。但今天，面对科技水平远超想象的新人类联盟，他终于不再隐藏。

"威廉，"林天头也不回地说，"去解除其他半机械人的控制。这艘战舰，我会守住的。"

威廉深深鞠了一躬："是！"

他转身冲出观景台，开始执行林天的命令。

而林天，则静静地站在落地窗前，等待着即将到来的战斗。

远处，天际线上，四个银色的小点正在快速接近。

那是新人类联盟的其他四艘战舰。

新人类联盟的真相，正在逐渐揭开。

而更大的风暴，即将来临。
