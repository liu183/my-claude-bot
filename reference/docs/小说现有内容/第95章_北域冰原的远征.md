# 第95章 北域冰原的远征

天宫浮空岛,起飞平台。

清晨的第一缕阳光穿透云层,洒落在整装待发的大军身上。三千名精锐战士身着银白色战甲,手持精钢武器,列成整齐的方阵。每一个战士身上都散发着强横的气息,最弱的也是一阶中期,最强的甚至达到了三阶初期。

这样的配置,足以横扫末世初期的任何势力。

林天站在高台上,俯视着这支强悍的军队。他的身边,龙灵儿、陈若曦、韩雪儿三位美女并肩而立。四人皆是战袍加身,英姿飒爽。

"诸位将士!"林天的声音通过异能放大,响彻整个起飞平台,"今天,我们将远征北域冰原,收复被丧尸占据的失地。那里的环境恶劣,敌人强大,但我相信,凭借你们的实力和勇气,定能凯旋而归!"

"万岁!"三千战士齐声高呼,声震云霄。

龙灵儿凑到林天耳边,娇笑道:"夫君,这次远征可是妾身第一次参与如此大规模的行动呢。妾身的雷系异能,定要那些冰原丧尸见识见识什么是天威!"

陈若曦也不甘示弱:"灵儿姐姐,你的雷系异能虽然强大,但到了冰原,妾身的冰系异能才是主场。到时候,可别让妾身抢了风头。"

韩雪儿温柔一笑:"两位姐姐别争了,妾身负责感知敌情和安抚伤员。夫君常说,团队配合才是最重要的。"

林天笑道:"你们说得都对。不过,此行最重要的是安全。三千精锐战士是天宫的根基,伤亡越小越好。"

他心中清楚,这次远征的目的不仅是收复失地,更是为了探索北域冰原的秘密。情报显示,那里隐藏着关于末世真相的线索。

"出发!"林天大手一挥。

浮空岛缓缓升空,朝着北方疾驰而去。三千精锐战士登上运输飞舟,紧随其后。庞大的舰队划破长空,气势磅礴。

......

三个小时后。

舰队越过大夏帝国的北域边境,进入了北域冰原的范围。

刹那间,温度骤降。从温暖的春意,直接跌至零下五十度的极寒。狂风夹杂着冰雪,呼啸而来。

"好冷!"龙灵儿打了个哆嗦,立刻运转异能抵抗寒意,"这冰原的温度,比妾身想象的还要低。"

陈若曦倒是如鱼得水,周身散发着淡淡的寒气:"妾身倒是觉得很舒服。冰系异能者对寒冷的适应性很强。"

韩雪儿突然神色一凝:"夫君,妾身感知到前方有大量生物反应。数量至少上万,而且……它们正在朝我们冲来!"

林天眼神一冷:"丧尸群。全军准备战斗!"

话音刚落,下方的冰原突然炸开。无数白色的身影从雪地中冲出,朝着上空的 fleet 嗷嗷怪叫。

这些丧尸已经适应了冰原的极寒环境。它们身上长出了厚厚的白色毛发,肌肉更加发达,速度比普通丧尸快了数倍。

"数量果然过万。"林天冷静分析,"一阶后期居多,二阶初期约有两千只。灵儿,若曦,该你们表现了。"

龙灵儿娇喝一声:"妾身这就来!"

她双手猛地一挥,紫色的雷霆瞬间从天而降。数道粗大的闪电劈入丧尸群中,每一次劈落都有数十只丧尸化为焦炭。

"雷霆万钧!"

陈若曦也不甘示弱,双手凝聚出无数冰锥:"冰锥暴雨!"

成千上万根冰锥如暴雨般射向丧尸群。那些冰锥锋利无比,轻易洞穿丧尸的头颅。

两只异能配合默契,不到十分钟,上万只丧尸全数被歼灭。

林天满意地点头:"不错,配合得很好。继续前进,我们的目标在更北面。"

舰队继续深入冰原。

......

又飞行了五个小时,舰队已经深入北域冰原五百公里。

这里的温度更低,达到了零下八十度。狂风呼啸,能见度极低。

"夫君,妾身感知到前方有人类反应。"韩雪儿突然说道,"而且……他们很害怕。"

林天精神一振:"人类?在这冰原深处?"

"是的,大约二十人。他们……不是普通人类,有点像半人半兽。"

"兽族?"林天心中一动。末世降临后,不少动物也发生了变异,进化出了智慧和类人形态。其中一些更是建立了自己的文明。

"全军降落!"林天下令。

运输飞舟缓缓降落在冰原上。林天带着三位美女和百名精锐战士走了下来。

前方不远处,二十名身材高大的生物正警惕地看着他们。

这些生物直立行走,身高超过两米,浑身覆盖着银白色的毛发。它们的头部保留着狼的特征,但身体却更加接近人类。手中握着骨制的长矛和战刀,看起来原始而凶猛。

狼族战士!

二十名狼族战士看到林天等人,眼中满是惊恐。它们低声咆哮,但不敢轻举妄动。

"别怕。"林天主动开口,"我们不是来攻击你们的。"

狼族战士们面面相觑,似乎对林天的话感到惊讶。

一名身材格外高大的雌性狼人走了出来。她有着银色的长发,俊美的狼脸透露出野性的魅力。身上的毛发并不浓密,反而勾勒出火辣的身材曲线。

"你们……不是尸皇的军队?"她用生硬的大夏语言问道。

林天摇头:"我们来自天宫,是来对抗丧尸的。"

狼族女战士松了口气,但依旧警惕:"天宫?妾身没听说过。不过,只要不是尸皇的人,就好。"

韩雪儿柔声说道:"你们为什么这么害怕?尸皇对你们做了什么?"

狼族女战士叹息道:"尸皇统治着整个北域冰原,我们兽族都是他的奴隶。每年,我们都要献祭一百名兽族美女,供尸皇享用。那些被送去的姐妹们,没有一个能回来。"

龙灵儿怒道:"太可恶了!你们为什么不反抗?"

"反抗?"狼族女战士苦笑,"尸皇是七阶巅峰的恐怖存在,我们兽族最强大的狮王也不过五阶初期,根本无法抗衡。"

林天心中一动。七阶巅峰,确实是强大的存在。但他如今也是六阶后期,加上三位娇妻和天宫的底蕴,未必没有一战之力。

"我是天宫之主林天。"林天直视对方,"如果我说,我们能帮你们推翻尸皇的统治,给你们自由,你们愿意相信吗?"

狼族女战士美眸一闪:"你……真的能打败尸皇?"

"当然。"林天自信道,"不仅是你,整个兽族都能获得自由。作为交换,我需要你们提供情报,并在关键时刻协助我们作战。"

狼族女战士沉吟片刻,突然单膝跪地:"妾身银月,愿意追随大人!只要能推翻尸皇的统治,妾身愿意做任何事!"

她抬起头,狼脸上带着动人的媚意:"大人若是喜欢,妾身今晚就可以侍奉您……"

陈若曦咳嗽一声:"咳咳,银月姑娘,我们可没说要让你侍奉。结盟是可以的,但夫君的后宫已经很拥挤了。"

龙灵儿也嗔道:"就是,想进我们天宫的后宫,可得先过我们这一关!"

银月愣了愣,随即展颜一笑:"原来几位姐姐也是大人的侍妾?妾身失礼了。不过,妾身对大人的仰慕是真诚的。若大人不嫌弃,妾身愿意成为大人的盟友,或者……更多。"

她说最后几个字时,眼神中带着明显的暗示。

林天装作没听见,问道:"银月,你是狼族的什么人?"

"妾身是狼族的首领之一,被称为狼王。"银月自豪道,"狼族在兽族联盟中排名第二,仅次于狮族。"

"很好。"林天点头,"带我们去见兽族联盟的其他首领。我想和他们谈谈结盟的事。"

银月欣喜道:"大人真的愿意帮助我们整个兽族?那太好了!请跟妾身来。"

她转身对其他狼族战士说了几句,那些战士立刻欢呼起来。

......

兽族联盟的营地位于一处巨大的冰洞中。

这里的温度比外面稍高,大约零下三十度。数百名兽族战士在营地中巡逻,看到银月带着林天等人到来,都露出好奇的目光。

"几位请稍等,妾身去召集其他首领。"银月说道。

不一会儿,三名强大的兽族走了过来。

第一名是一名雄狮族战士,身高接近三米,浑身肌肉虬结,金色的鬃毛威风凛凛。他散发的气息极其恐怖,赫然达到了五阶初期。

狮王雷蒙!

第二名自然就是银月,四阶后期的实力在兽族中已经算顶尖强者。

第三名是一名熊族战士,体型同样庞大,浑身黑毛如铁。他散发的气息略逊于银月,但也是四阶后期。

熊王泰山!

"天宫之主林天。"狮王雷蒙打量着林天,"银月说你能打败尸皇,给我们自由?这话我听过很多次了,大多数都是骗子。"

林天平静道:"我是不是骗子,很快就会证明。狮王,如果你愿意结盟,我们就能联手推翻尸皇。否则,天宫也会独自行动,只是届时兽族能否获得自由,就难说了。"

熊王泰山瓮声瓮气道:"大人若是真能打败尸皇,我熊族愿意追随!"

狮王雷蒙沉默片刻,突然笑道:"好!我狮族也同意结盟!"

林天敏锐地捕捉到雷蒙眼中一闪而过的异样光芒,但他没有点破。

"很好。"林天道,"那么,我们结盟了。天宫三千精锐,加上兽族五千战士,总兵力八千。我们的计划是,兽族做内应,我们做外合,里应外合,突袭尸皇宫殿!"

银月兴奋道:"太好了!我们终于有希望了!"

她看着林天,眼中的爱慕更加明显。

林天心中暗笑。这次远征,不仅收获了强大的盟友,还意外收获了狼族美女的芳心。不过,后宫确实有点拥挤了,改天得和灵儿她们好好商量商量。

"准备战斗。"林天下令,"目标——尸皇宫殿!"

冰原的风雪中,一支八千人的联军蓄势待发。他们即将面临的,是七阶巅峰的尸皇,以及整个北域冰原最恐怖的丧尸军团。

胜利,就在眼前!
