# 第97章 空间法则的威力

轰——！

巨大的爆炸声在宫殿内回荡，林天被一道恐怖的能量波震退了七八步，每一步都在地面上留下深深的脚印。

尸皇悬浮在半空中,黑色的死亡能量在他周身翻涌,如同活物般律动。他嘴角勾起一抹冷笑:"就这点实力吗？我还以为你能给我带来什么惊喜。"

林天稳住身形,擦了擦嘴角的血迹,眼中却没有任何恐惧,反而闪烁着诡异的光芒:"惊喜？你还没看到我真正的实力。"

"哦？"尸皇双眸微眯,"难道你还有什么底牌不成？"

林天没有回答,只是缓缓抬起右手。刹那间,一股恐怖的能量波动从他体内爆发而出,整个宫殿的空间开始剧烈扭曲。

"这是——"尸皇脸色骤变,他感受到了一股前所未有的法则之力。

"空间法则！"尸皇惊骇地吼道,"这不可能！你不过是个九阶,怎么可能掌握法则之力？"

林天冷笑:"没什么不可能的。在这个世界上,只有想不到,没有做不到。"

随着林天的话音落下,他身后的空间开始剧烈扭曲,形成了一个巨大的黑色漩涡。漩涡中仿佛蕴含着吞噬天地的恐怖力量,连周围的光线都被吞噬进去。

"去！"林天挥手,一道透明的空间刃从漩涡中飞出,看似不起眼,却蕴含着恐怖的切割之力。

尸皇大惊,连忙闪避。但那道空间刃的速度快到极致,瞬间就到了他面前。

"不——！"尸皇怒吼,浑身爆发出黑色的死亡能量,试图阻挡空间刃。

嗤——！

空间刃轻易地穿透了死亡能量屏障,在尸皇的左臂上划出一道深深的伤口。鲜血喷涌而出,但奇怪的是,伤口处没有血液流出,反而不断有灰色的死气溢出。

"这怎么可能？"尸皇看着自己手臂上的伤口,眼中满是不可置信,"空间法则竟然能无视防御？"

"空间法则的本质就是切断空间。"林天淡淡道,"任何防御在空间面前,都如同虚设。"

尸皇的脸色变得狰狞起来:"好！很好！我倒要看看,你的空间法则能坚持多久！"

话音落下,尸皇体内爆发出更加恐怖的能量波动。黑色的死亡能量如潮水般涌出,瞬间笼罩了整个宫殿。与此同时,他体内的冰系异能也被激发到极致,寒气席卷四周。

"冰封天下！"尸皇怒吼。

刹那间,整个宫殿被厚厚的冰层覆盖,所有的一切都被冻成了冰雕。就连空气都仿佛凝固了,温度骤降至零下百度。

林天也被冻在了一块巨大的冰晶中,看上去如同琥珀中的昆虫。

"哈哈！"尸皇狂笑,"空间法则又如何？还不是被我冰封了？现在,我要把你碎尸万段！"

他抬手打出一道能量波,准备将冰晶中的林天粉碎。

然而——

咔嚓——！

冰晶突然裂开了一道缝隙。

尸皇的笑容僵在脸上:"什么？"

咔嚓咔嚓——！

裂缝越来越多,最终,整个冰晶彻底碎裂。林天从中走出,身上连一丝冻伤都没有。

"这——这怎么可能？"尸皇彻底震惊了,"我的冰封之力,就算是九阶强者也难以抵挡,你怎么可能——"

"你以为,我会让你随便封印吗？"林天淡淡道,"早在冰封之前,我就已经用空间之力将自己转移到了空间夹缝中。你看到的,不过是我留下的残影罢了。"

"空间夹缝？"尸皇的脸色变得更加难看。

"没错。"林天一步步向前,"空间法则的运用,远比你想象的要复杂。不仅能切割,还能转移,更能——"

他突然停下脚步,双眸中闪烁着诡异的光芒:"禁锢！"

刹那间,尸皇周围的空间开始剧烈扭曲,仿佛被无形的力量锁死。他惊恐地发现,自己竟然无法移动分毫,就连体内的能量流动都变得极其缓慢。

"这——这是什么？"尸皇惊骇地吼道。

"空间禁锢。"林天淡淡道,"将你周围的空间彻底锁死,让你成为瓮中之鳖。"

"不！这不可能！"尸皇拼命挣扎,但无济于事。他的身体就像被钉在了原地,连一根手指都无法移动。

"没什么不可能的。"林天走到尸皇面前,平静地看着他,"你输了。"

"我不服！"尸皇愤怒地吼道,"我可是十阶尸皇!怎么可能输给你这个九阶——"

"九阶又如何？"林天打断他的话,"实力的差距,不是用阶位来衡量的。法则之力,你可以理解为另一种形式的力量。"

他抬起右手,一道巨大的空间刃在掌心凝聚而成,足有数十米长,散发着令人心悸的毁灭气息。

"再见。"林天淡淡道。

"不——！"尸皇发出绝望的吼叫。

嗤——！

巨大的空间刃劈下,尸皇的身体瞬间被切成两半。鲜血喷涌,但奇怪的是,血液中没有任何生命力,反而充满了浓烈的死气。

"咳咳......"被切成两半的尸皇还没有彻底死亡,他艰难地抬起头,看着林天,眼中竟然露出了一抹诡异的笑容。

"你以为......结束了吗？"尸皇虚弱地说道。

林天眉头微皱:"什么意思？"

"真正的......主人......要来了......"尸皇的声音越来越小,"你......你会后悔的......"

"真正的主人？"林天追问道,"你说的是谁？观测者？还是其他什么存在？"

但尸皇已经无法回答了。他的身体开始快速消散,最终化为一堆灰烬,消散在空气中。

林天站在原地,若有所思。

尸皇临死前的话,让他心中升起一股不安。真正的主人？这个尸皇,居然也是被人操控的傀儡？

如果是这样,那么那个操控尸皇的存在,实力绝对远超尸皇。至少,也是掌握了法则之力的强者,甚至可能是——

十一阶？

林天深吸一口气,将这个念头压下。不管真正的主人是谁,现在考虑这些还为时过早。当务之急,是继续提升实力,直到能应对任何威胁。

他转身,朝着宫殿出口走去。

刚走出宫殿,一群人就从四面八方围了上来。为首的,正是唐柔、林婉儿和洛冰霜。

"林天！"唐柔第一个冲了过来,扑进他怀里,"你没事吧？我们刚才感受到恐怖的能量波动,担心死了!"

"我没事。"林天拍了拍她的背,温柔地说道,"尸皇已经解决了。"

"真的？"林婉儿惊喜道,"太好了!"

洛冰霜也松了口气:"那我们现在怎么办？"

林天看向远方,眼中闪烁着坚定的光芒:"继续前进。这个末世中,还有太多的秘密等待我们去揭开。"

他顿了顿,又补充道:"而且,我们还需要变得更强。因为真正的敌人,可能还没有出现。"

众人一愣,随即郑重地点了点头。

夕阳西下,余晖洒在众人身上,拉出长长的影子。而在那遥远的未来,更多的挑战和危险,正在等待着他们。

但无论如何,林天都知道,他不会退缩。因为他有要守护的人,有要追寻的真相。

而这一切,都需要足够的力量。

空间法则,不过是他力量的开始。未来的路还很长,而他,将继续前行。
