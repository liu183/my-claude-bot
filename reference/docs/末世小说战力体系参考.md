## 📈 五、升级条件和方式

### 基础升级条件

```
【一阶异能者】
├─ 觉醒条件: 末世降临后基因突变
├─ 晋升二阶: 击杀10只以上一阶丧尸 OR 吸收1颗一阶晶核
└─ 预计时间: 1-2周

【二阶异能者】
├─ 晋升三阶: 击杀50只以上二阶丧尸 OR 吸收5颗二阶晶核
├─ 预计时间: 1-2个月
└─ 关键: 掌握领域雏形

【三阶异能者】
├─ 晋升四阶: 击杀三阶丧尸领主 OR 吸收10颗三阶晶核
├─ 预计时间: 3-6个月
└─ 关键: 形成完整领域

【四阶异能者】
├─ 晋升五阶: 击杀四阶君王丧尸 OR 吸收高阶能量
├─ 预计时间: 6-12个月
└─ 关键: 掌握法则雏形

【五阶异能者】
├─ 晋升六阶: 吸收星能核心 OR 突破极限
├─ 预计时间: 1-2年
└─ 关键: 掌握完整法则

【六阶异能者】
├─ 晋升七阶: 未知条件
├─ 预计时间: 未知
└─ 关键: 掌握多重法则
```

### 特殊升级方式

1. **基因强化**: 通过沈冰的基因强化计划,可快速提升实力
   - 第一批: 陈若曦、冷月突破至四阶初期
   - 第二批: 龙灵儿突破至四阶后期

2. **星能吸收**: 吸收外星能量,可跨越式提升
   - 林天吸收星能核心,直接从五阶突破至六阶

3. **系统奖励**: 完成系统任务,获得奖励
   - 解锁新能力
   - 获得特殊物品
   - 提升魅感指数

4. **战斗突破**: 生死战斗中突破极限
   - 击杀强大敌人
   - 激发潜能
   - 领悟新能力

## 📊 八、战力评估标准

### 综合战力评估公式

```
综合战力 = 基础星力 × 异能倍率 × 装备倍率 × 战术倍率

【基础星力】
├─ 1星 = 10点战力
├─ 5星 = 50点战力
├─ 10星 = 100点战力
├─ 15星 = 150点战力
└─ 20星 = 200点战力

【异能倍率】
├─ 普通异能: ×1.0
├─ 元素系异能: ×1.2
├─ 特殊系异能: ×1.5
├─ 稀有系异能: ×1.8
└─ 法则级异能: ×2.0

【装备倍率】
├─ 无装备: ×1.0
├─ 普通装备: ×1.1
├─ 精良装备: ×1.3
├─ 史诗装备: ×1.5
└─ 传说装备: ×2.0

【战术倍率】
├─ 无战术: ×1.0
├─ 基础战术: ×1.2
├─ 高级战术: ×1.5
├─ 大师级战术: ×1.8
└─ 完美战术: ×2.0
```

### 实战评估案例

#### 案例1: 四星巅峰林天 vs 四阶中期暴君
```
林天战力 = 12 × 1.5(空间) × 1.0 × 1.5 = 27点
暴君战力 = 11 × 1.0(力量) × 1.3 × 1.0 = 14.3点
战力差距: 1.89倍
结果: 林天险胜(付出代价)
```

第10章: 林天三星初期
第15章: 林天三星中期

```

(逻辑合理)
```

```
错误示例:
"林天是五星巅峰"
"林天是六阶初期"
(混用阶和星)

正确示例:
统一使用"六阶(16星)"或"五星巅峰(15星)"
(统一标准)
```

## 🎯 附录:常用术语

### 战力相关术语

- **星数**: 异能者实力的基础计量单位,1-21星
- **阶数**: 异能者实力的等级划分,一阶至七阶
- **领域**: 高阶异能者能控制的特殊空间
- **法则**: 传说级异能者能掌握的宇宙规则
- **晶核**: 丧尸体内的能量结晶,可用于提升实力
- **星能**: 外星能量,威力远超普通异能

### 角色相关术语

- **魅感**: 角色的魅力指数,影响收服难度
- **忠诚度**: 角色对林天的忠诚程度
- **等级**: 角色的美貌等级(S级、SS级、SSS级)
- **定位**: 角色在团队中的角色定位
