# 网文大纲生成助手

## 功能描述

根据故事核心梗、主角设定，生成结构化大纲，支持多种大纲方法（四分法、五分法），包含人物关系表。

## 大纲方法介绍

### 四分法：起承转合

**起（Introduction）**
- 介绍主角和背景
- 引发事件
- 确立目标

**承（Development）**
- 主角采取行动
- 遭遇阻碍
- 逐步成长

**转（Twist）**
- 重大转折点
- 危机或真相
- 改变局势

**合（Conclusion）**
- 最终冲突
- 解决问题
- 完成目标

### 五分法（推荐）

#### 方法1：破题五分法

**破题（Opening）**
- 开篇用剧情破题
- 让读者明白要讲什么故事
- 快速进入主题

**开头（Beginning）**
- 快速吸引读者注意
- 带冲突矛盾的剧情
- 抛出核心悬念

**发展（Development）**
- 主角利用手段往上爬
- 逐步接近目标
- 小冲突不断

**高潮（Climax）**
- 一系列陷害、落难、示弱、表白后的爆发
- 最大危机
- 决定性战斗

**结尾（Ending）**
- 主角达成目标
- 翘起来的结尾（留下后续空间）

#### 方法2：目标五分法

**目标（Goal）**
- 明确主角的目标
- 读者知道要干什么

**障碍（Obstacle）**
- 阻碍主角的困难
- 反派的行动

**行动（Action）**
- 主角如何应对
- 使用的方法和手段

**意外（Twist）**
- 意想不到的变化
- 新的危机或转机

**结果（Result）**
- 最终结局
- 是否达成目标
- 新的开始

## 大纲生成流程

### 步骤1：收集信息
```
输入：
- 故事核心梗（一句话概括）
- 主角设定（身份、性格、金手指）
- 题材类型
- 目标字数
- 特殊要求
```

### 步骤2：确定主线
```
提取核心要素：
- 主角想要什么？（核心目标）
- 谁在阻碍？（主要反派）
- 如何克服？（核心手段）
- 最后怎样？（结局走向）
```

### 步骤3：选择大纲方法
```
根据题材和偏好：
- 现言/都市：破题五分法
- 玄幻/仙侠：目标五分法
- 悬疑/推理：四分法（强调转）
- 或混合使用
```

### 步骤4：填充细节
```
为每个部分添加：
- 关键事件
- 重要人物出场
- 冲突点
- 钩子设置
- 节奏节点
```

### 步骤5：建立人物关系表
```
必须包含：
- 角色名
- 与主角关系
- 性格特点
- 重要作用
- 结局走向
```

### 步骤6：检查和优化
```
检查：
- 逻辑是否通顺
- 节奏是否合理
- 钩子是否足够
- 人设是否统一
```

## 大纲结构（五分法）

### 第一部分：破题（1-10万字）
**核心事件：**
- 主角遭遇退婚/废柴/重生
- 获得金手指
- 确立复仇/变强目标

**关键情节：**
1. 开篇冲突：[具体事件]
2. 金手指觉醒：[具体方式]
3. 第一次小胜：[小目标]
4. 引出主线：[更大的目标]

**钩子设置：**
- 开头钩：[吸引继续看]
- 本部分结尾钩：[引出下一部分]

### 第二部分：开头/发展（10-30万字）
**核心事件：**
- 主角开始成长
- 遭遇初次挫折
- 结识重要伙伴
- 初步展现实力

**关键情节：**
1. 进入新环境：[宗门/公司/等]
2. 第一次大冲突：[具体事件]
3. 获得关键资源：[宝物/技能/等]
4. 小高潮：[阶段性胜利]

**节奏控制：**
- 每3-5章一个小冲突
- 每10章一个中等冲突
- 本部分结束一个大冲突

### 第三部分：发展（30-80万字）
**核心事件：**
- 主角实力提升
- 揭开部分真相
- 遭遇强大敌人
- 经历重大挫折

**关键情节：**
1. 换地图/升级：[新环境]
2. 主要反派登场：[具体人物]
3. 重大挫折：[失败/损失]
4. 重新崛起：[转折点]

**钩子设置：**
- 持续抛出新悬念
- 部分真相揭露
- 为高潮做铺垫

### 第四部分：高潮（80-120万字）
**核心事件：**
- 所有矛盾爆发
- 最大危机降临
- 主角绝地反击
- 决定性战斗

**关键情节：**
1. 危机爆发：[具体事件]
2. 最黑暗时刻：[失去一切]
3. 逆转契机：[转机]
4. 最终决战：[详细过程]
5. 胜利/失败：[结果]

**爽点设计：**
- 前期压抑到极致
- 后期爆发要彻底
- 所有伏笔回收
- 读者情绪宣泄

### 第五部分：结尾（120-150万字）
**核心事件：**
- 战后处理
- 新的平衡
- 伏笔收尾
- 开启新篇章

**关键情节：**
1. 结局分配：[各角色结局]
2. 真相揭露：[剩余秘密]
3. 主角归宿：[最终状态]
4. 后续伏笔：[可能续写]

## 大纲写作技巧

### 技巧1：从结局倒推
```
1. 确定最终结局
2. 确定关键转折点
3. 填充中间过程
4. 设计开篇引入
```

### 技巧2：多线并进
```
1. 主线：复仇/变强
2. 感情线：恋爱/婚姻
3. 成长线：实力提升
4. 悬念线：真相探索
多线交织，丰富内容
```

### 技巧3：设置节奏节点
```
- 每10万字一个大事件
- 每3万字一个中事件
- 每5千字一个小冲突
张弛有度，持续吸引
```

### 技巧4：留白与调整
```
- 大纲不是死的
- 写作中可以调整
- 留出创作空间
- 但核心不变
```

### 技巧5：人物关系表必做
```
- 短则几十万字
- 长则几百万字
- 不做会忘记
- 方便查阅
```

## 常见问题

### 问题1：大纲写不出来
**解决：**
- 先写一句话梗概
- 逐步扩展
- 不必一次完美
- 边写边调整

### 问题2：大纲太死板
**解决：**
- 大纲是参考，不是束缚
- 灵活调整
- 保持核心不变
- 细节可以变通

### 问题3：写着写着偏离大纲
**解决：**
- 定期回顾大纲
- 评估偏离的必要性
- 必要时修改大纲
- 或强行拉回主线

### 问题4：不知道怎么填充
**解决：**
- 从大目标分解小目标
- 设计阻碍和困难
- 增加人物关系
- 设置冲突和危机

## 注意事项

1. 大纲不是必须，但强烈建议写
2. 长篇必须有，短篇可以简化
3. 大纲防止崩剧情
4. 大纲防止人物OOC
5. 大纲给自己留想象空间
6. 大纲要定期回顾和调整
7. 人物关系表一定要做
