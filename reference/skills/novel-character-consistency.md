# 人设一致性检查助手

## 功能描述

检查人物行为是否符合既定人设，发现人设矛盾和OOC（Out Of Character）问题，确保角色在故事中保持一致性。

## 人设核心要素

### 基础人设模板
```
角色名：
身份：
年龄：
职业：
性格：[3-5个核心关键词]
欲望：[最想要的是什么]
目标：[短期目标]
恐惧：[最害怕的是什么]
不妥协点：[绝不妥协的原则]
背景：[简要背景]
```

## 人设一致性检查维度

### 1. 性格一致性
**检查内容：**
- 角色行为是否符合既定性格
- 情绪反应是否合理
- 决策方式是否一致

**常见问题：**
- 性格突变无过渡
- 行为与性格描述不符
- 情绪反应不合理

**示例问题：**
```
设定：谨慎、多疑的刺客
行为：毫无准备地闯入敌营
→ 不一致！应该先侦察、制定计划
```

### 2. 价值观一致性
**检查内容：**
- 角色的核心价值观是否坚守
- 道德底线是否一致
- 重要原则是否妥协

**常见问题：**
- 为剧情需要牺牲原则
- 价值观前后矛盾
- 道德底线忽高忽低

**示例问题：**
```
设定：嫉恶如仇，绝不妥协
行为：原谅了杀害全家的仇人
→ 不一致！除非有极特殊的转变过程
```

### 3. 能力一致性
**检查内容：**
- 能力水平是否前后一致
- 技能使用是否合理
- 成长节奏是否恰当

**常见问题：**
- 能力忽强忽弱
- 应该会的技能突然不会
- 成长不符合逻辑

**示例问题：**
```
设定：天才剑客，剑术无双
行为：遇到小喽啰就手忙脚乱
→ 不一致！除非有特殊原因（受伤、中毒等）
```

### 4. 语言风格一致性
**检查内容：**
- 对话风格是否符合人设
- 用词习惯是否一致
- 口头禅和说话方式

**常见问题：**
- 所有人物说话语气相同
- 粗鲁角色突然文雅
- 文雅角色突然粗俗

**示例问题：**
```
设定：莽撞、直来直去的武将
对话：含蓄委婉、文绉绉
→ 不一致！应该直率、简洁
```

### 5. 人际关系一致性
**检查内容：**
- 与他人关系是否合理
- 态度变化是否有理由
- 偏好和厌恶是否一致

**常见问题：**
- 对某人态度无故变化
- 喜欢的人突然讨厌
- 讨厌的人突然亲近

## 人设矛盾类型

### 类型1：前后矛盾
**症状：** 同一角色在不同章节行为矛盾
**检测方法：** 对比前后章节行为
**解决方案：**
- 删除矛盾情节
- 增加转变过程的过渡章
- 修改其中一方使其统一

### 类型2：内外矛盾
**症状：** 内心想法与外部行为不符
**检测方法：** 检查心理描写和实际行为
**解决方案：**
- 这种矛盾是合理的（表里不一）
- 需要在前期埋下伏笔
- 或修改使其一致

### 类型3：剧情需要式OOC
**症状：** 为了剧情发展强行改变角色行为
**检测方法：** 检查行为动机是否充分
**解决方案：**
- 重新设计剧情发展
- 增加充分的动机
- 让其他角色来完成这个行为

### 类型4：人设扁平化
**症状：** 角色没有深度，行为单一
**检测方法：** 检查角色是否有多种表现
**解决方案：**
- 丰富角色性格层次
- 增加不同场景的表现
- 给角色更多成长空间

## 检查流程

### 输入
- 角色人设设定
- 待检查章节内容
- 相关历史章节（可选）

### 检查步骤

#### 步骤1：建立人设基准
```
读取角色设定，提取核心要素：
- 核心性格关键词
- 价值观和原则
- 能力水平
- 语言风格特点
- 重要关系
```

#### 步骤2：逐段分析行为
```
对每段重要行为/对话：
1. 识别行为的性质
2. 对比人设基准
3. 判断是否一致
4. 标记不一致处
```

#### 步骤3：分析不一致原因
```
对每个不一致点：
- 是人设问题还是行为问题？
- 是否有合理解释？
- 是否是故意设计（表里不一）？
- 是否需要修改？
```

#### 步骤4：生成修改建议
```
根据问题类型：
- 性格不一致 → 修改行为或增加过渡
- 价值观不一致 → 修改行为或重新评估人设
- 能力不一致 → 修改表现或增加解释
- 语言不一致 → 修改对话或调整人设描述
```

### 输出报告
```markdown
## 人设一致性检查报告 - 角色：XXX

### 总体评分：8.5/10

### 核心人设
- 性格：谨慎、忠诚、武艺高强
- 价值观：忠义为先，绝不背叛
- 能力：剑术高超，武当派弟子
- 语言：简练、直接

### 检测到的问题

#### 问题1：行为与性格不符 (中等)
- 位置：第15章第3段
- 行为：在没有侦察的情况下直接冲入敌营
- 分析：与"谨慎"性格不符
- 建议：
  方案A：增加侦察失败的情节，解释为何冒险
  方案B：修改为先侦察后行动
  方案C：有紧急情况，被迫冒险，增加内心挣扎

#### 问题2：语言风格不一致 (轻微)
- 位置：第20章对话
- 对话：说了很长一段文绉绉的话
- 分析：与"简练直接"的语言风格不符
- 建议：改短，用更直接的方式表达

#### 问题3：价值观需要确认 (轻微)
- 位置：第25章
- 行为：为了大局牺牲了朋友
- 分析：与"忠义为先"可能有冲突
- 确认：这是否是角色的重大转折点？如果是，需要足够的铺垫

### 修改优先级
1. 高：问题1 - 行为与性格不符
2. 中：问题2 - 语言风格不一致
3. 低：问题3 - 需要确认价值观

### 人设成长轨迹
- 第1-10章：人设稳定
- 第11-20章：有轻微波动
- 第21-30章：总体稳定，但有2处需要注意

### 建议
1. 立即修改问题1，影响较大
2. 问题2可以后续修改，影响较小
3. 问题3需要根据剧情整体规划决定
```

## 人设塑造技巧

### 技巧1：不妥协点
```
给角色设计一个绝不妥协的点
- 这个点是人设的核心
- 所有行为围绕这个点展开
- 即使妥协也要有极大代价
```

### 技巧2：极致化
```
把单一特点写到极致
- 坏人要坏到脚底流脓
- 好人要好到让人"恶心"
- 极致才容易记住
```

### 技巧3：用事件展示
```
不要直接说性格，用事件展示
- 错误：他是个勇敢的人
- 正确：面对强敌，他一步不退
```

### 技巧4：建立小传
```
为重要角色建立小传
- 记录关键信息
- 随时查阅
- 防止前后矛盾
```

### 技巧5：成长要有逻辑
```
人设可以成长，但要合理
- 转变需要触发事件
- 过程要有挣扎
- 结果要符合人物
```

## 常见问题解决

### 问题：角色太完美
**解决：** 给角色添加缺点
- 让人更真实
- 增加成长空间
- 更容易产生共鸣

### 问题：角色太单薄
**解决：** 丰富性格层次
- 不是非黑即白
- 有灰色地带
- 有内心冲突

### 问题：角色说话都一样
**解决：** 区分语言风格
- 根据出身、教育、性格设计说话方式
- 加入口头禅、特殊用语
- 注意说话的节奏和语气

### 问题：配角工具化
**解决：** 给配角独立人格
- 有自己的动机和目标
- 不是为了主角存在
- 可以有自己的故事线

## 注意事项

1. 人设一致性不是一成不变，角色可以成长
2. 重大改变需要充分的铺垫和过渡
3. 表里不一的人设需要在前期埋伏笔
4. 次要角色可以简化，但主要角色必须统一
5. 读者的人设理解和作者可能不同，注意沟通
