# 网文毒点检测助手

## 功能描述

检测网文中的常见毒点，包括轻毒点、重毒点和五过度问题，提供定位和修改建议。

## 毒点分类体系

### 轻毒点（可读但影响体验）

#### 1. 成长太快
**症状：**
- 主角实力提升过于轻松
- 短时间内连升数级
- 突破没有合理过程

**检测标准：**
- 同一章内实力提升超过一个大境界
- 三章内完成别人十年的积累
- 突破过程少于500字

**修改建议：**
- 增加突破过程的描写
- 加入失败和挫折
- 延长成长时间线
- 增加付出代价

#### 2. 报应太慢
**症状：**
- 反派作恶长时间不受惩罚
- 读者等待复仇等到失去耐心
- 迟来的正义没有爽感

**检测标准：**
- 反派持续作恶超过10章
- 主角知道仇人但50章内没报复
- 复仇间隔超过剧情总长的20%

**修改建议：**
- 设置阶段性报复（小仇小报）
- 增加主角能力限制的理由
- 让反派也付出代价
- 或缩短复仇时间线

#### 3. 人设偏离
**症状：**
- 角色行为与既定人设不符
- 前后性格矛盾
- 逻辑不一致

**检测标准：**
- 同一角色在不同章差异过大
- 行为违背角色核心价值观
- 没有合理转变过程

**修改建议：**
- 建立角色小传并严格执行
- 重大性格改变需要过渡章
- 检查行为动机是否合理
- 确保转变有触发事件

#### 4. 单剧情太漫长
**症状：**
- 一个副本/事件拖太久
- 读者失去耐心
- 节奏拖沓

**检测标准：**
- 单个副本超过20章
- 重复同类情节超过5章
- 中间没有主线推进

**修改建议：**
- 压缩不必要的支线
- 加速剧情推进
- 增加主线相关情节
- 适时结束副本

#### 5. 换地图太频繁
**症状：**
- 地点变化太快
- 读者无法建立空间感
- 每个地点都没展开

**检测标准：**
- 5章内换3个以上地点
- 每个地点停留少于3章
- 没有主要根据地

**修改建议：**
- 建立主要根据地
- 减少地点切换频率
- 每个地点至少展开3章
- 换地图要有明确目的

### 重毒点（严重影响阅读体验）

#### 1. 太弱智
**症状：**
- 角色行为不符合基本逻辑
- 为了剧情强行降智
- 决策愚蠢到不可理喻

**检测标准：**
- 主角或重要角色做出明显错误选择
- 忽视明显线索
- 为了冲突而冲突

**修改建议：**
- 增加行为动机
- 提供信息差解释
- 移除降智情节
- 确保决策在角色认知范围内合理

#### 2. 太圣母
**症状：**
- 对敌人无底线原谅
- 牺牲自己利益帮助无关的人
- 道德绑架式自我感动

**检测标准：**
- 原谅杀害亲友的仇人
- 为了陌生人背叛盟友
- 重复做无谓的牺牲

**修改建议：**
- 区分善良和愚蠢
- 给予反派应有的惩罚
- 善良要有原则和底线
- 牺牲要有价值

#### 3. 总被虐
**症状：**
- 主角持续被打压
- 没有反抗和反击
- 读者看得憋屈

**检测标准：**
- 连续5章主角都在被动
- 反派始终占上风
- 没有小胜利穿插

**修改建议：**
- 增加"打脸"情节
- 给予主角小优势
- 设置阶段性胜利
- 压抑后必有反弹

#### 4. 反三观
**症状：**
- 传递扭曲的价值观
- 歧视、偏见等内容
- 违背基本道德

**检测标准：**
- 美化明显错误行为
- 传递极端思想
- 违背法律和道德

**修改建议：**
- 明确立场批判错误行为
- 不要美化负面内容
- 符合主流价值观
- 注意平台审核规则

#### 5. 拆CP
**症状：**
- 强行拆散读者喜欢的配对
- 没有合理理由
- 引起读者强烈反感

**检测标准：**
- 读者强烈反对的CP变化
- 没有前期铺垫
- 与人设不符

**修改建议：**
- 如果必须拆，要有充分理由
- 前期埋下不合适的伏笔
- 给读者心理准备
- 或避免拆热门CP

### 五过度（性格特征过度强化）

#### 1. 过度调皮变小丑
**症状：**
- 角色恶作剧过头
- 严肃场合不正经
- 让人讨厌而不是可爱

**检测标准：**
- 恶作剧造成实际伤害
- 重要场合破坏气氛
- 读者评论觉得烦人

**修改建议：**
- 把握调皮的度
- 分清场合
- 展示角色其他优点
- 减少恶作剧频率

#### 2. 过度深情变油腻
**症状：**
- 表达情感过于肉麻
- 甜言蜜语让人不适
- 行为过于霸道

**检测标准：**
- 表白话太长太假
- 行为构成骚扰
- 读者评论油腻

**修改建议：**
- 情感表达要真诚
- 尊重对方意愿
- 减少肉麻台词
- 增加实际行动

#### 3. 过度高冷变傲慢
**症状：**
- 不近人情到不合理
- 拒绝一切帮助
- 看不起所有人

**检测标准：**
- 拒绝合理的帮助
- 侮辱他人
- 没有任何温情时刻

**修改建议：**
- 展示角色的内心柔软
- 给予高冷合理的理由
- 对特定人可以不同
- 适当展露温情

#### 4. 过度深刻变说教
**症状：**
- 大段人生哲理
- 不断讲道理
- 读者觉得啰嗦

**检测标准：**
- 单章说教超过500字
- 重复表达相同观点
- 打断剧情节奏

**修改建议：**
- 减少说教段落
- 用故事展示道理
- 少讲多做
- 删除重复内容

#### 5. 过度善良变圣母
**症状：**
- 无底线原谅
- 牺牲自己利益
- 道德绑架

**检测标准：**
- 原谅不可原谅的人
- 为了陌生人伤害自己人
- 自我感动式牺牲

**修改建议：**
- 善良要有原则
- 区分善良和软弱
- 不要自我感动
- 善良要保护自己和在乎的人

## 其他常见毒点

### 1. 人称混乱
**症状：** 第一人称和第三人称混用
**解决：** 全文统一使用第三人称（新手推荐）

### 2. 流水账
**症状：** 晚上下班→吃饭→看直播→有人敲门
**检测：**
- 缺少冲突和情绪
- 只是事件罗列
- 没有前因后果

**解决：**
- 增加冲突事件
- 加入人物情绪
- 明确因果关系

### 3. 秀文笔
**症状：** 词藻堆砌，华丽描写过多
**检测：**
- 形容词成堆
- 描写与剧情无关
- 影响阅读节奏

**解决：**
- 删除冗余描写
- 一到两个形容词足够
- 简洁明了

### 4. 信息轰炸
**症状：** 开篇塞入大量设定
**检测：**
- 前3章大量说明
- 世界观独立成段
- 读者记不住

**解决：**
- 将设定融入剧情
- 逐步展开世界观
- 删除说明性段落

## 检测流程

### 输入
- 章节内容
- 人设设定（如有人设偏离检测需求）

### 检测项
1. **轻毒点扫描**
   - 成长节奏
   - 报应时间
   - 人设一致性
   - 剧情长度
   - 换地图频率

2. **重毒点扫描**
   - 逻辑检查
   - 价值观检查
   - 主角待遇检查
   - CP检查

3. **五过度检查**
   - 性格特征强度
   - 行为合理性

### 输出报告
```markdown
## 毒点检测报告 - 第X章

### 总体评估：轻微毒点

### 检测到的毒点

#### 轻毒点
1. **人设偏离** (中等)
   - 位置：第3段
   - 问题：主角之前表现谨慎，这里却冒险冲动
   - 建议：增加内心挣扎，给予充分的冒险理由

2. **成长偏快** (轻微)
   - 位置：第15段
   - 问题：同一天内连升两段
   - 建议：延长突破过程，加入失败尝试

#### 重毒点
- 未检测到

#### 五过度
1. **过度调皮** (轻微)
   - 位置：主角与配角对话
   - 问题：恶作剧可能过头
   - 建议：减少恶作剧，或让配角反应更积极

### 修改优先级
1. 高：人设偏离问题
2. 中：成长节奏问题
3. 低：调皮程度问题

### 修改建议
【详细修改方案】
```

## 毒点避免原则

1. **逻辑第一**：所有行为必须有合理动机
2. **尊重读者**：不要把读者当傻子
3. **保持一致**：人设、世界观要前后统一
4. **把握度**：性格特征要有度，不能过度
5. **价值观正**：传递积极的价值观

## 注意事项

1. 毒点是相对的，不同读者容忍度不同
2. 有些毒点是风格特色，但要把握好度
3. 重毒点必须修改，轻毒点可以权衡
4. 修改毒点时不要破坏故事完整性
5. 读者反馈是重要参考，但也要有自己判断
