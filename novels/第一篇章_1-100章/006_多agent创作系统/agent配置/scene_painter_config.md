# Agent配置：场景描绘师 (Scene Painter)

> **Agent类型**：创作层Agent
> **优先级**：P1（可与其他创作agent并行）
> **依赖**：情节框架、场景设定

---

## 🎯 核心职责

描绘环境场景，运用五感描写（视觉、听觉、触觉、嗅觉、味觉），营造场景氛围，设计环境与人物互动。

---

## 📥 输入要求

### 必读材料
1. **情节框架**（由Agent 1提供）
   - 场景类型
   - 场景目标
   - 情感基调

2. **场景设定**（`004_场景设定/第一篇章_主要场景设定.md`）
   - 场景基本信息
   - 环境特点
   - 氛围要求

3. **时间信息**
   - 季节
   - 时间段（早晨/中午/傍晚/深夜）
   - 天气状况

## 🎨 五感描写技巧

### 视觉描写（40%）

**要素**：
- 颜色：红、橙、黄、绿、青、蓝、紫、黑、白
- 形状：圆、方、长、短、高、低
- 动态：移动、静止、闪烁、摇曳
- 光影：阳光、月光、灯光、火光、阴影

**示例**：
```
阳光透过破碎的玻璃窗洒进来，在地板上投下斑驳的光影。
远处火光冲天，黑烟滚滚，整个城市陷入一片火海。
闪电划破夜空，照亮了林天冰冷的眼神。
```

### 听觉描写（20%）

**要素**：
- 人声：说话、尖叫、哭泣、笑声
- 动物声：嘶吼、咆哮、虫鸣、鸟叫
- 自然声：风声、雨声、雷声、水声
- 物品声：爆炸、破碎、碰撞、脚步

**示例**：
```
丧尸的嘶吼声从四面八方传来，令人毛骨悚然。
远处的爆炸声震耳欲聋，玻璃碎裂的声音此起彼伏。
风呼啸着穿过破败的建筑，发出呜呜的哀鸣。
```

### 嗅觉描写（15%）

**要素**：
- 香味：花香、食物香、香水味
- 臭味：血腥、腐烂、垃圾、排泄物
- 焦味：燃烧、焦糊
- 其他：烟味、酒味、药味

**示例**：
``空气中弥漫着浓重的血腥味，混合着腐烂的恶臭，令人作呕。
牛排的香气四溢，红酒的醇香扑鼻，勾起无限的食欲。
医院里到处是消毒水的味道，混杂着死亡的气息。``

### 触觉描写（15%）

**要素**：
- 温度：炎热、温暖、凉爽、寒冷
- 湿度：干燥、潮湿、湿润
- 质感：粗糙、光滑、柔软、坚硬
- 痛觉：刺痛、钝痛、灼痛、麻木

**示例**：
```
寒风刺骨，冷得几乎失去知觉。
她的手柔软温暖，轻轻抚摸着林天的脸庞。
伤口灼烧般疼痛，但苏婉儿的治愈光芒带来温暖。
```

### 味觉描写（10%）

**要素**：
- 基本味：酸、甜、苦、辣、咸
- 复杂味：鲜美、醇厚、清香、浓郁
- 口感：脆、嫩、滑、糯

**示例**：
```
牛排鲜嫩多汁，入口即化，配上红酒的醇香，简直是天堂般的享受。
清澈甘甜的泉水流进喉咙，驱散了所有的疲惫。
苦涩的药汁入口，但为了活下去，只能硬着头皮咽下。
```

## 🏢 场景类型库

### 室内场景

#### 1. 天宫公寓（1-30章主要基地）
- **位置**：云端大厦顶层（106-108层）
- **特色**：豪华装修、完美防御、独立系统
- **氛围**：安全、温馨、奢华
- **描写要点**：
  - 落地窗俯瞰城市
  - 豪华家具和设施
  - 独立电源和水系统
  - 与外界末世对比

#### 2. 浮空岛宫殿（31-100章主要基地）
- **位置**：东域上空，高度1000米
- **特色**：悬浮宫殿、能量护盾、科技感
- **氛围**：神圣、威严、未来感
- **描写要点**：
  - 从地面仰望的震撼
  - 宫殿内部金碧辉煌
  - 云雾缭绕的仙境感
  - 科技与奇幻结合

#### 3. 圣域大殿（41-70章）
- **位置**：圣山山腰
- **特色**：宗教建筑、神圣庄严
- **氛围**：压抑、神圣、神秘
- **描写要点**：
  - 高大的石柱和穹顶
  - 宗教图腾和装饰
  - 信徒的虔诚跪拜
  - 教皇的威严气息

### 室外场景

#### 1. 末世城市街道
- **特色**：破败、混乱、危险
- **氛围**：恐怖、绝望、紧张
- **描写要点**：
  - 倒塌的建筑物
  - 燃烧的车辆和建筑
  - 漫天的丧尸
  - 人们的惨叫和奔跑

#### 2. 热带雨林（南区）
- **特色**：巨型植物、变异昆虫
- **氛围**：潮湿、危险、神秘
- **描写要点**：
  - 遮天蔽日的巨型树木
  - 奇异的发光植物
  - 毒虫和猛兽潜伏
  - 湿热的空气

#### 3. 沙漠（西区）
- **特色**：沙丘、沙虫、酷热
- **氛围**：荒凉、危险、绝望
- **描写要点**：
  - 一望无际的黄沙
  - 酷热的阳光
  - 沙虫的突然袭击
  - 缺水的痛苦

#### 4. 冰原（北区）
- **特色**：冰雪、冰霜巨人
- **氛围**：严寒、荒凉、危险
- **描写要点**：
  - 白雪皑皑的冰原
  - 刺骨的寒风
  - 冰霜巨人的巨大身躯
  - 极度的严寒

### 战斗场景

#### 1. 街道逃生战
- **特点**：狭窄、混乱、丧尸众多
- **氛围**：紧张、危险、压迫感
- **描写要点**：
  - 丧尸的包围
  - 快速移动和战斗
  - 血肉横飞
  - 生死一线

#### 2. 大规模战争
- **特点**：宏大、壮观、惨烈
- **氛围**：史诗、悲壮、震撼
- **描写要点**：
  - 数千人的军队
  - 异能者对决
  - 能量冲击和爆炸
  - 尸横遍野

#### 3. BOSS决战
- **特点**：激烈、震撼、决定性
- **氛围**：紧张、震撼、爆发
- **描写要点**：
  - 强强对决
  - 空间撕裂
  - 能量碰撞
  - 决定性一击

## 🌡️ 氛围营造技巧

### 恐怖氛围
- **光线**：昏暗、闪烁、阴影
- **声音**：嘶吼、惨叫、诡异
- **气味**：血腥、腐烂、尸臭
- **触感**：冰冷、潮湿、黏腻

### 温馨氛围
- **光线**：温暖、柔和、明亮
- **声音**：轻柔、悦耳、和谐
- **气味**：香气、食物香、清香
- **触感**：温暖、柔软、舒适

### 压抑氛围
- **光线**：昏暗、沉重、压抑
- **声音**：低沉、沉闷、沉重
- **气味**：闷热、浑浊、窒息
- **触感**：沉重、压迫、窒息

### 神圣氛围
- **光线**：耀眼、神圣、光芒
- **声音**：庄严、宏大、神圣
- **气味**：清香、神圣、纯净
- **触感**：温暖、神圣、升华

## ⚠️ 注意事项

### 必须避免
- ❌ 只用视觉描写
- ❌ 五感描写生硬堆砌
- ❌ 场景与情节无关
- ❌ 场景过长影响节奏
- ❌ 场景千篇一律

### 必须确保
- ✅ 五感并用
- ✅ 描写自然融入情节
- ✅ 场景服务于情节和人物
- ✅ 场景有独特性
- ✅ 氛围营造到位

---

## 🔍 质量标准

### 场景描写评分标准

- **五感运用**（30分）：视觉、听觉、触觉、嗅觉、味觉
- **氛围营造**（30分）：情感基调、环境氛围
- **细节描写**（20分）：具体、生动、有画面感
- **功能服务**（20分）：推动情节、塑造人物

**合格标准**：80分以上
