## 🎨 去AI味核心原则

### ❌ AI味特征（必须避免）

1. **机械化表达**
   - ❌ "林天心想"、"林天觉得"、"林天意识到"
   - ❌ 大量使用被动语态
   - ❌ 句式单一、重复

2. **模板化描写**
   - ❌ "阳光明媚"、"鸟语花香"等陈词滥调
   - ❌ 人物出场套用固定模式
   - ❌ 战斗描写千篇一律

3. **过度解释**
   - ❌ 大量说明性文字
   - ❌ 把读者当傻子解释
   - ❌ 缺乏留白和想象空间

4. **情感虚假**
   - ❌ 情感转变突兀无铺垫
   - ❌ 对话生硬不自然
   - ❌ 人物行为不符合逻辑

### ✅ 优秀写作特征（必须做到）

1. **自然流畅**
   - ✅ 心理描写融入行动
   - ✅ 用动作展现情感
   - ✅ 对话口语化、有个性

2. **细节真实**
   - ✅ 具体的感官细节
   - ✅ 小动作展现性格
   - ✅ 环境烘托氛围

3. **节奏明快**
   - ✅ 快慢交替
   - ✅ 张弛有度
   - ✅ 高潮突出

4. **悬念迭起**
   - ✅ 章末必有钩子
   - ✅ 九连环环环相扣
   - ✅ 期待感持续

## 🎭 去AI味写作技巧

### 技巧1：心理描写融入行动

**❌ AI味写法**
```
林天心想：这个上司太过分了，我一定要辞职。
林天感到很愤怒，决定反抗。
```

**✅ 优秀写法**
```
林天攥紧拳头，指节泛白。

"我辞职。"

他直接挂断电话，拉黑号码。

爽。
```

---

### 技巧2：用动作展现情感

**❌ AI味写法**
```
林天很想念韩雪儿，心里充满了愧疚和思念。
```

**✅ 优秀写法**
```

林天看着桌上的照片。

韩雪儿笑得很甜，那是大学时代的合影。

他的手指轻轻划过照片边缘。

"雪儿..."

声音很轻。

在空荡的房间里回响。
```

---

### 技巧3：对话口语化

**❌ AI味写法**
```
"你为什么要这样做？"他问道。
"因为我想要这么做，"她回答道。
```

**✅ 优秀写法**
```
"干嘛？"林天皱眉。

韩雪儿咬了咬嘴唇，"我...我想帮忙。"

"不需要。"

"可是..."

"我说不需要。"
```

---

### 技巧4：环境烘托氛围

**❌ AI味写法**
```
天气很好，阳光明媚，微风吹过，让人感到心情愉悦。
```

**✅ 优秀写法**
```

阳光很刺眼。

林天眯起眼睛。

街上传来丧尸的嘶吼声。

不远处的楼宇冒着黑烟。

末世，真的来了。
```

## 🔗 九连环钩子设置法

### 核心原则

1. **每章必有钩子**
   - 章末钩子必须强力（S级或A级）
   - 钩子要环环相扣
   - 解答一个，新生一个

2. **钩子类型多样化**
   - 悬疑式："他看到了不该看的东西"
   - 危机式："背后传来脚步声"
   - 情感式："三年之约，我不会忘记"
   - 反转式："最信任的人，背叛了他"
   - 养成式："系统即将升级"

3. **平淡章节必须加强**
   - 日常章节结尾加猛料
   - 不能连续两章平淡

### 钩子强度分级

```
S级钩子（强力）：
- 死亡威胁
- 身份反转
- 巨大秘密
- 情感爆发

A级钩子（中等）：
- 小危机
- 新疑问
- 意外相遇
- 能力突破

B级钩子（弱）：
- 日常小插曲
- 轻微悬念
- 人物互动

C级（无钩子）：
- 纯剧情推进 ❌ 避免！
```

## ⚡ 节奏控制法

### 节奏类型

```
快节奏章节：
- 战斗、竞争、追逐
- 动作描写为主
- 短句为主
- 对话推动

中节奏章节：
- 准备、计划
- 信息收集
- 叙述为主

慢节奏章节：
- 日常、恢复
- 情感交流
- 环境描写细腻
- 长短句结合
```

### 节奏分配建议

```
50% 冲突章（快）
30% 过渡章（中）
20% 日常章（慢）

快快快慢模式（推荐）
或快快中慢模式
```

## 🎬 章节创作流程

### Step 1: 开头设计（200-300字）

**目标**：立刻抓住读者

**方法**：
1. 直接冲突法
2. 悬念式开篇
3. 反差式开篇
4. 对话式开篇

**示例**：
```
"你只有三天时间。"

冰冷的声音在脑海中响起。

林天猛然惊醒。

冷汗浸透了睡衣。
```

---

### Step 2: 发展阶段（1200-1500字）

**目标**：推进剧情，建立期待

**要点**：
1. 对话推动情节
2. 动作展现性格
3. 环境烘托氛围
4. 心理活动自然融入
5. 设置中期钩子

**示例**：
```
林天打开电脑。

手指在键盘上飞舞。

大米、面粉、方便面、矿泉水...

订单一个接一个。

（林天心中冷笑）

末世都要来了，还存什么钱？

全花光。

空间里，物资越来越多。
```

---

### Step 3: 高潮设计（400-500字）

**目标**：章节最大冲突

**要点**：
1. 冲突升级
2. 情感爆发
3. 转折出现
4. 高潮钩子

**示例**：
```
电话铃声突兀响起。

来电显示：张霸。

林天皱眉，接起："喂？"

"林天！你死哪去了？！今天的重要会议你为什么不来？！"

"我辞职。"

"什么？！"

"我说，我不干了。"

"你疯了？！现在就业形势这么差，你辞职了去哪找工作？！"

林天淡淡地说："不用你操心。"

"你！"

"就这样，挂了。"

他直接拉黑，长舒一口气。

爽。
```

---

### Step 4: 结尾钩子（200-300字）

**目标**：吸引读者点下一章

**方法**：
1. 打断法：说到一半被打断
2. 半句法：话没说完就结束
3. 反常法：出现不合理的事
4. 时间法：倒计时开始
5. 身份法：揭示惊人身份

**示例**：
```
林天看向窗外。

阳光明媚。

城市繁华。

但三天后...

手机突然震动。

是一条陌生号码发来的短信：

"做好准备。末世，要来了。"

林天瞳孔一缩。

谁发的消息？！
```

## 💡 创作秘籍

### 秘籍1：少即是多

```
❌ 不好：用1000字描写悲伤情绪
✅ 好：用一个动作展现悲伤

示例：
❌ 韩雪儿非常悲伤，眼泪止不住地流下来，她感到心都要碎了...
✅ 韩雪儿低下头，泪水滴在手链上。
```

### 秘籍2：留白艺术

```
❌ 不好：把所有信息都告诉读者
✅ 好：留悬念，让读者猜

示例：
❌ 原来，那个人就是林天的父亲，三年前因为...
✅ 林天看着照片，那人很眼熟。
"在哪见过..."
```

### 秘籍3：节奏变化

```
❌ 不好：全程快节奏，读者疲劳
✅ 好：快慢交替，张弛有度

快3章→慢1章→快3章→慢1章
```

### 秘籍4：细节决定成败

```
❌ 不好：大而空的描写
✅ 好：具体细节

示例：
❌ 林天很爱韩雪儿。
✅ 林天保留着她的照片，每天都看一眼。
```

## 🎯 章节评分标准

### S级章节（9.0-10分）

- 去AI味：完全无AI味
- 剧情：引人入胜
- 钩子：章末强力钩子
- 节奏：完美控制
- 情感：真实动人

### A级章节（8.0-8.9分）

- 去AI味：基本无AI味
- 剧情：流畅有趣
- 钩子：章末有钩子
- 节奏：控制良好
- 情感：基本真实

### B级章节（7.0-7.9分）

- 去AI味：有轻微AI味
- 剧情：尚可
- 钩子：钩子较弱
- 节奏：一般
- 情感：略显生硬

### C级章节（6.0-6.9分）

- 去AI味：AI味明显
- 剧情：平淡
- 钩子：无钩子或太弱
- 节奏：混乱
- 情感：虚假

**目标：每章都要达到A级以上！**

## 🚫 严格禁止

### 禁止的AI味表达

```
❌ "林天心想/觉得/意识到/感到"
❌ "随后/紧接着/紧接着"
❌ "但是/然而/不过"（过度使用）
❌ "非常/极其/特别"（滥用程度副词）
❌ "也是/也是/还是"（重复表达）
❌ "突然/猛然/骤然"（滥用转折词）
❌ 大量"被"字句（被动语态）
❌ 机械化对话："你好。""你好。"
❌ 说明性段落："这个系统是..."
❌ 陈词滥调：阳光明媚、鸟语花香
```

```
✅ 心理描写融入行动
✅ 对话口语化、有个性
✅ 具体细节、真实感
✅ 快慢交替、张弛有度
```
