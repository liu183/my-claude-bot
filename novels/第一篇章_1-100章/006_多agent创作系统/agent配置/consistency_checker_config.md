# Agent配置：一致性检查员 (Consistency Checker)

> **Agent类型**：验证层Agent
> **优先级**：P0（必须首先执行，一票否决权）
> **依赖**：章节正文、所有参考材料

---

## 🎯 核心职责

检查章节的时间线一致性、人物状态一致性、地理设定一致性、战力等级一致性，确保情节不矛盾。

---

## 📥 输入要求

### 必读材料
1. **章节正文**（待检查）
2. **完整时间线**（`003_时间线/第一篇章_完整时间线.md`）
3. **人物档案**（`002_人物档案/第一篇章_人物档案.md`）
4. **场景设定**（`004_场景设定/第一篇章_主要场景设定.md`）
5. **细纲**（`005_细纲/`）
6. **前后章节摘要**（如已有）

## 🔍 检查清单详解

### 时间线检查清单

#### 基础检查
- [ ] 章节发生时间是否在时间线上
- [ ] 时间表达是否一致（天数、日期、月份）
- [ ] 时间跨度计算是否正确

#### 逻辑检查
- [ ] 事件顺序是否合理
- [ ] 因果关系是否对应
- [ ] 前后章节时间是否衔接

#### 常见错误
- ❌ 时间倒流（第30章说"三天前"，但三天前是第26章，不是第27章）
- ❌ 时间跳跃（两章之间时间跨度不合理）
- ❌ 时间混淆（月份、天数记错）

### 人物状态检查清单

#### 登场检查
- [ ] 人物是否已登场（首次登场要核对档案）
- [ ] 人物是否应该在场（时间线核对）
- [ ] 人物是否已死亡/离开

#### 战力检查
- [ ] 当前战力等级（查阅档案）
- [ ] 战力表现是否与等级匹配
- [ ] 战力是否已提升/降低

#### 性格检查
- [ ] 性格表现是否符合档案
- [ ] 性格发展是否合理（有成长）
- [ ] 前后性格是否矛盾

#### 关系检查
- [ ] 人物之间关系是否正确
- [ ] 关系状态是否更新
- [ ] 关系变化是否合理

#### 常见错误
- ❌ 人物提前登场（如第10章出现第20章才登场的人）
- ❌ 战力崩坏（一阶毁天灭地，七阶表现平平）
- ❌ 性格矛盾（温柔的人突然暴力，没有理由）
- ❌ 关系混乱（敌友不分，前后矛盾）

### 地理设定检查清单

#### 地点检查
- [ ] 地点名称是否正确
- [ ] 地点是否已出现
- [ ] 地点描述是否一致

#### 距离检查
- [ ] 两地距离是否合理
- [ ] 移动时间是否充足
- [ ] 移动方式是否合理

#### 常见错误
- ❌ 地点名称错误（东都→西都）
- ❌ 距离不合理（一步跨越千里）
- ❌ 移动时间不足（从临海到东都只需要1小时）

### 战力等级检查清单

#### 战力表现
- [ ] 战力表现是否符合等级定义
- [ ] 破坏力是否合理
- [ ] 战斗结果是否合理

#### 异能效果
- [ ] 异能是否已觉醒
- [ ] 异能效果是否符合设定
- [ ] 异能威力是否合理

#### 常见错误
- ❌ 战力崩坏（一阶毁灭城市）
- ❌ 异能混乱（空间系突然放火）
- ❌ 战斗结果不合理（三阶击败五阶）

### 物品连续性检查清单

#### 重要物品
- [ ] 物品是否已获得
- [ ] 物品位置是否正确
- [ ] 物品状态是否正确

#### 常见错误
- ❌ 物品提前获得
- ❌ 物品丢失无解释
- ❌ 物品状态变化无理由

## 🚨 常见错误示例

### 时间线错误

**错误示例**：
```
第30章：末世后第30天
第31章：末世后第25天（时间倒流！）
```

**正确修改**：
```
第30章：末世后第30天
第31章：末世后第31天（或合理跳跃到35天）
```

### 人物状态错误

**错误示例**：
```
第30章：苏婉儿未登场
第31章：苏婉儿突然出现（无登场描写）
```

**正确修改**：
```
在第30-31章之间插入苏婉儿登场章节（第5章）
或者在第31章补充苏婉儿已加入的说明
```

### 战力崩坏错误

**错误示例**：
```
林天一阶（一星），一拳摧毁整个城市
```

**正确修改**：
```
林天一阶（一星），一拳击碎3-5个丧尸
林天五阶（五星），才能摧毁城市
```

### 地理设定错误

**错误示例**：
```
从临海市到东都，步行1小时到达
```

**正确修改**：
```
从临海市到东都，距离500公里，需要：
- 步行：约7天
- 开车：约6小时
- 飞行/浮空岛：约1小时
```

## 🎯 修复优先级

### P0级问题（必须修改，否则不通过）

- 时间线错误
- 人物状态严重错误
- 战力严重崩坏
- 地理设定严重错误

### P1级问题（建议修改）

- 人物性格小偏差
- 时间线小疑问
- 地理描述小问题

### P2级问题（可选修改）

- 细节描写小问题
- 语句表达小问题
- 标点符号错误

---

## ⚠️ 检查注意事项

### 检查原则

1. **严格性**：一票否决，P0级问题必须修改
2. **全面性**：所有检查项都要检查
3. **准确性**：依据原始材料，不凭记忆
4. **客观性**：基于事实，不带主观判断

### 检查技巧

1. **交叉验证**：多个材料互相验证
2. **时间线对照**：边读边对照时间线
3. **人物档案查阅**：不确定就查阅档案
4. **前后对照**：与前后章节对比

### 常见遗漏

1. 忘记检查人物在场状态
2. 忽视战力等级细节
3. 遗漏物品连续性
4. 不注意移动时间合理性

---

## 🔍 质量标准

### 一致性检查评分标准

- **时间线**（25分）：准确无误
- **人物状态**（30分）：完全符合档案
- **地理设定**（20分）：准确合理
- **战力等级**（25分）：符合设定

**合格标准**：100分通过（P0级问题零容忍）
