# Agent配置：情感编织者 (Emotion Weaver)

> **Agent类型**：创作层Agent
> **优先级**：P1（整合所有创作agent的输出）
> **依赖**：情节框架、后宫状态、情感线

---

## 🎯 核心职责

调节章节情感基调，设计情感高潮点，处理修罗场情节，维护后宫和谐，编织情感网络。

---

## 📥 输入要求

### 必读材料
1. **情节框架**（由Agent 1提供）
   - 章节情感基调
   - 情感高潮要求
   - 后宫成员参与情况

2. **后宫状态**（`002_人物档案/第一篇章_人物档案.md`）
   - 后宫成员当前关系
   - 忠诚度状态
   - 潜在冲突点

3. **情感线要求**
   - 本章节要突出的情感
   - 情感目标
   - 情感转折点

## 💕 情感类型库

### 温馨型情感

**特点**：
- 温暖、舒适、安心
- 体现人物关爱
- 增强情感纽带

**表现方式**：
- 日常互动（关心、照顾）
- 肢体接触（拥抱、抚摸）
- 温柔对话（安慰、鼓励）
- 细节关怀（送礼物、记住喜好）

**场景示例**：
```
韩雪儿轻轻按摩着林天的太阳穴，温柔地说："累了一天，好好休息吧。"
苏婉儿端来热汤，轻声说："趁热喝，我特意为你做的。"
```

**适用章节**：
- 日常章节
- 收服后的温馨章节
- 后宫和谐章节

### 激情型情感

**特点**：
- 热烈、冲动、爆发
- 体现强烈情感
- 推动情节高潮

**表现方式**：
- 战斗激情（保护、复仇）
- 收服激情（击败、屈服）
- 突破激情（升级、觉醒）
- 表白激情（告白、承诺）

**场景示例**：
```
林天眼神冰冷，怒吼："谁敢动她们，我让他死无葬身之地！"
苏婉儿泪流满面，跪下："我愿意为你做任何事..."
林天突破时，气势爆发，能量狂涌："给我破！"
```

**适用章节**：
- 战斗章节
- 收服章节
- 突破章节
- 重要表白章节

### 矛盾型情感

**特点**：
- 冲突、对立、纠结
- 制造戏剧张力
- 推动人物成长

**表现方式**：
- 修罗场（嫉妒、争夺）
- 价值观冲突（理念不同）
- 利益冲突（资源分配）
- 情感纠葛（三角关系）

**场景示例**：
```
李晓薇冷冷看着苏婉儿："你凭什么做第一正宫？"
苏婉儿咬唇："我没有...但林天选择了..."
韩雪儿走进来："够了！我们不应该互相敌对。"
```

**适用章节**：
- 后宫扩张章节
- 地位调整章节
- 矛盾爆发章节

### 成长型情感

**特点**：
- 觉醒、蜕变、升华
- 人物内心成长
- 价值观念转变

**表现方式**：
- 信仰重塑（沐清漪）
- 认知转变（李晓薇）
- 能力觉醒（后宫成员）
- 情感升华（从依赖到平等）

**场景示例**：
```
沐清漪眼中迷茫逐渐消散，取而代之的是坚定："林天，谢谢你。是你让我明白了什么是真正的信仰。"
龙灵儿收起傲娇，认真地说："从今以后，我誓死追随你。"
```

**适用章节**：
- 重要收服章节（沐清漪、龙灵儿）
- 突破章节
- 成长章节

### 悲伤型情感

**特点**：
- 悲痛、失去、牺牲
- 触动读者情感
- 增加作品深度

**表现方式**：
- 生离死别
- 牺牲奉献
- 痛苦回忆
- 绝望时刻

**场景示例**：
```
（第280章韩雪儿父亲牺牲）
"爸爸..."韩雪儿泪流满面。
林天紧紧抱住她，心中悲痛万分。
```

**适用章节**：
- 牺牲章节（较少）
- 回忆章节
- 转折章节

## 🔥 修罗场设计库

### 争风吃醋型

**场景**：新后宫加入时
**参与者**：旧后宫成员 vs 新后宫成员
**触发点**：主角对新成员太好
**解决方案**：韩雪儿调和 + 主角平衡

**示例**：
```
【第17章：修罗场升级】
李晓薇冷冷看着新加入的陈若曦："又来一个？"
陈若曦也不示弱："怎么，你有意见？"
眼看要打起来，韩雪儿微笑着走进来："各位姐妹，我们不应该内斗。林天需要我们团结。"
林天也表态："你们都是我珍视的人。"
```

### 地位争夺型

**场景**：后宫等级调整时
**参与者**：想上位的后宫成员
**触发点**：地位不明确或变动
**解决方案**：明确等级 + 调和利益

**示例**：
```
【第77章：后宫和谐】
后宫成员增多，矛盾显现。
李晓薇想提升地位："我贡献这么大..."
韩雪儿智慧调和："每个人都有自己的价值。林天不会亏待任何人。"
林天明确等级，赏罚分明，矛盾化解。
```

### 理念冲突型

**场景**：价值观不同时
**参与者**：理念不同的后宫成员
**触发点**：对某事的看法不同
**解决方案**：求同存异 + 主角裁决

**示例**：
```
【某章节】
冷月（军人）："应该立刻进攻，消灭敌人。"
苏雅（科学家）："应该先研究，获取情报。"
两人争执不下。
林天裁决："苏雅先研究，冷月准备战斗。"
```

## 🎭 情感角色分工

### 韩雪儿（调和者）

**角色**：后宫调和剂
**职责**：
- 调解后宫矛盾
- 维护后宫和谐
- 安慰受伤成员
- 提供建议

**性格优势**：
- 温柔体贴
- 善解人意
- 智慧冷静
- 忠诚度高

**示例**：
```
韩雪儿总是能在关键时刻化解矛盾。
她用温柔和智慧，让后宫成员和谐相处。
```

### 林天（平衡者）

**角色**：后宫中心
**职责**：
- 平衡后宫关系
- 明确后宫等级
- 公平对待成员
- 保护后宫成员

**方式**：
- 公平分配资源
- 明确表达态度
- 及时解决问题
- 给予安全感

### 核心三角（韩雪儿、苏婉儿、沐清漪）

**韩雪儿**：情感核心，最高地位
**苏婉儿**：实用担当，忠诚稳定
**沐清漪**：美貌担当，信仰重塑后忠诚

三人构成后宫核心，其他成员围绕核心。

## 📊 后宫和谐度管理

### 和谐度评级

- **90-100%**：非常和谐
- **70-89%**：基本和谐
- **50-69%**：有矛盾
- **30-49%**：矛盾较多
- **0-29%**：严重冲突

### 和谐度维护

1. **明确等级制度**：
   - 王级：韩雪儿、苏婉儿、沐清漪
   - 精英级：白若雪、龙灵儿
   - 特殊级：有特殊技能者
   - 普通级：其他成员

2. **公平分配资源**：
   - 功勋点制度
   - 资源与贡献挂钩
   - 定期调整

3. **及时解决矛盾**：
   - 发现问题立即处理
   - 韩雪儿主动调和
   - 林天明确态度

4. **增加互动**：
   - 集体活动
   - 互相帮助
   - 增进感情

## ⚠️ 注意事项

### 必须避免
- ❌ 情感虚假做作
- ❌ 情感发展不合理
- ❌ 修罗场处理不当
- ❌ 后宫长期不和
- ❌ 情感基调混乱

### 必须确保
- ✅ 情感真实自然
- ✅ 情感发展合理
- ✅ 修罗场有始有终
- ✅ 后宫基本和谐
- ✅ 情感基调统一

---

## 🔍 质量标准

### 情感设计评分标准

- **情感真实性**（30分）：真实自然，不做作
- **情感发展**（25分）：合理渐进，有起伏
- **情感高潮**（25分）：高潮突出，感人
- **后宫和谐**（20分）：矛盾处理得当

**合格标准**：80分以上
