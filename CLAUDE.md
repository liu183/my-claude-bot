# Novel Chapter Analyzer

## 项目概述
- **项目名称**: 网文章节分析器
- **分析对象**: D:\迅雷下载\筛检版 目录下的都市神医、乡村神医类网文
- **分析范围**: 每部小说前100章
- **分析内容**: 
  - 剧情主线梳理
  - 支线剧情分析
  - 后宫人物设计
  - 后宫剧情线
  - 创作风格分析
  - 创作素材模板生成

## 项目配置
- **主工作目录**: `D:\workspace\happy-claude-code\novel-analysis-chapter-analyzer`
- **数据源目录**: `D:\迅雷下载\筛检版`
- **分析输出**: `analysis-results/` 目录下的各小说分析报告

## 技术栈
- Python 3.9+
- Claude Code for project management
- Multi-agent architecture for parallel analysis

## 核心功能
1. 文本预处理与章节分割
2. 多Agent并行章节分析
3. 全局剧情结构化梳理
4. 人物关系网络分析
5. 创作风格特征提取
6. 素材模板自动生成
